Featured Project

SmartWrite

SmartWrite is an innovative note-taking application currently in development with a group of 6 other students. We aim to create a tool to help students and professors alike in creating more organized and effective notes.

The application implements handwriting recognition and AI-assisted writing features to enhance note-taking efficiency. We're designing a comprehensive organization system for managing and categorizing academic notes.

Through this project, I've gained valuable experience with GitHub workflows, version control, and CI/CD pipelines for automated testing and deployment. We utilize agile development practices to coordinate with team members and meet project milestones.
